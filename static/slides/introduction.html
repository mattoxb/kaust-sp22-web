Class Structure

  • Topics are divided by week.
    • Monday: 2 hour class to introduce content.
      • Short lecture
      • Group activities
      • Live coding demo
    • Thursday: 1 hour
      • Practical application
        • Instructor will demo a program
        • This will turn in to starter code for a project or assignment

Assignments and Grading

  • Most weeks will have a small homework set.
    • Due one week after assigned (usually in class the next week).
    • Feedback and opportunity for revision if good effort
  • There will be 2 &ndash; 4 longer assignments: “projects”
    • This will depend on your interests, in part!
    • Projects count as 3 homework sets.
  • We will use a standard 90/80/70 point standard for grades.
