Numerics Types

Python has different data types.

Integers
Basically, whole numbers. They can get as big as you like.
Floating Point

Floating point numbers.

  • The computer only keeps enough memory to store maybe 10 digits, after that it

truncates the value.

  • The decimal point is “floating” in that the number can be something like 12345.67 or 0.000001234567

Strings and Booleans

  • Short for “string of characters”.
  • You can just think of it as text.
  • Strings must be surrounded by either single quotes or double quotes.
  • Use + to append to strings together (called concatenating).
    • "H " + "9123" == "H 9123"
    • Use str to convert something into a string.
      • str(123) = "123"
    • Use int to convert a string into an integer.
      • int("123") = 123
      • So… "123"+"300"=="123300" but int("123") + int("300") == 423

Variables

Variable names begin with a lower case letter. After that there can be as many letters (upper and lower case), numbers, and certain special characters like _, ', etc. Each variable can keep track of one value.

  • Assign to a variable by using a single equal: x = 23
  • Compare two variables by using a double equal: x == y
    • Common mistake in languages is to put = instead of ==.
    • Does Python check?
  • We will talk about “scoping” in a few minutes.

Lists

  • Place values between square brackets, like [1,2,3]. You can put anything in a list, including other lists, and even functions.
  • You can access a specific element of a list or a string, where zero is the first element.
  • Note that double equals means “is equal to”, whereas a single equal means to assign something. Recall the Variables section.
  • Accessing a single element:
    • "Hello"[0] == "H"
    • [2,4,6,7][3] == 7

List Slices

  • Accessing a “slice”, e.g. [a:b]. It will give you the elements from a upto but not including b.
    • "Hello"[1:2] == "H"
    • "Hello"[1:4] == "Hel" I guess you need to be careful with this example.
  • Accessing elements from the end, use negative numbers.
    • "Hello"[-1] == "o"
  • It works with slices too.
    • "Hello"[3:-1] == "lo"

Dictionaries

  • Keeps a set of keys and corresponding values (like a word and its definition.)
  • Use curly braces to create them: phones = { "Jenni":8675390, "Emergency": 911 }
  • Use lookup syntax like this:
    • phones['Jenni'] == 8675309
  • There are some special functions:
    • phones.keys() == ['Jenni','Emergency']
    • phones.values() == [8675309,911]
    • Note that the order of the keys and values could be anything.

Looping

You can use a for loop to have a variable have many different values. 

for x in ["Hello","There","Peeps"]:
    print(x)

If Statements

You can have a “condition” and a “then branch” (what to do if the condition is true): 

x = 10
y = 20
if x > y:
    print("Hello")
if x < y:
    print("There")
print("Done")

If/Else Statements

You can also have an “else branch” to say what to happen if the condition is false. 

if x > y:
    print("Greater")
else:
    print("Not greater")

Chains of If Statements

You can chain if statements together useing elif (short for “else if”): 

if x>y:
    print("Greater")
elif x==y:
    print("Equal")
else:
    print("Less than")

   return a + b - 2

Variables in Functions

  • It’s possible to have multiple variables with the same name.
  • If you assign to a variable in the function, it creates a brand new variable with that name.
    • This is called a local variable
  • If you use a variable in a function, but don’t assign to it, it uses a top-level variable of that name.
    • This is called a global variable.
  • You can use the nonlocal keyword if you want to assign to a global variable. This is bad style though.
a = 10
b = 20
c = 30

def foo(x):
    nonlocal c
    b = 5
    y = a + b
    c = c + y
    return x + c

  • a is global
  • there are two variables b, one local, one global
  • there is one variable c.

If you run foo(10), then you will have:

  • a == 10
  • b == 20 (not 5!)
  • c == 30 + 10 + 5 == 45
  • The return value of the function will be 55.
