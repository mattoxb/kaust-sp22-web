For loops

Another kind of loop that is very common is the for loop. It has four parts!

  • An initialization
  • A conditional
  • An increment
  • A body.

Here is a for loop that computes the factorial function like last time. 

1: double fact(int n) {
2:   double r = 1;
3:   for(int i=1; i<=n; ++i) {
4:     r = r * i;
5:   }
6:   return r;
7: }

For loops are often used with arrays. 

 1: int stuff[100];
 2: int i;
 3: 
 4: cout << "How many elements are there? ";
 5: cin >> n;
 6: 
 7: for(i=0; i<n; ++i) {
 8:  cin >> stuff[i];
 9: }
10: 
11: double avg = 0;
12: 
13: for(i=0; i<n; ++i) {
14:   avg += stuff[i];
15: }
16: 
17: cout << "The sum is " << avg << " and the average is " << avg / n << endl;

A couple of things to notice:

  • The condition is a strict less than! In C++, arrays start with 0, and go up to size-1.

If you use less-than-equal you will go off the edge of the array and cause problems.

  • You can create a variable in the for loop itself (e.g., the int i=0 part). The variable only exists in the loop, and disappears afterwards.

Vectors

These C-style arrays are very primitive, and have no error checking. C++ introduces an improved array called a vector. It does bounds-checking, can grow automatically, and has many convenient functions built in. 

 1: vector<int> stuff;
 2: int i,x;
 3: 
 4: cout << "How many elements are there? ";
 5: cin >> n;
 6: 
 7: for(i=0; i<n; ++i) {
 8:  cin >> x;
 9:  stuff.push_back(x);
10: }
11: 
12: double avg = 0;
13: 
14: for(i=0; i<n; ++i) {
15:   avg += stuff[i];
16: }
17: 
18: cout << "The sum is " << avg << " and the average is " << avg / n << endl;
19: cout << "THere are " << stuff.size() << " elements in the vector." << endl;

The type is different. You say vector<int> to indicate the vector contains integers. This will make more sense when we get to objects and classes.

Strings

A string in C++ uses the type string, and you can use it as if it were a vector of characters. 

 1: #include <iostream>
 2: 
 3: using namespace std;
 4: 
 5: string foo;
 6: 
 7: int main() {
 8:   foo.push_back('a');
 9:   foo.push_back('b');
10:   foo.push_back('c');
11:   foo.push_back('d');
12:   std::cout << foo << endl;
13: }

This prints out abcd.
