The purpose of this guide is to help you get started with C++ quickly, for example so that you can I am going to skip quite a few subtleties as a result, but you can pick these up later if you want. I will assume that you are a beginning programmer. If you are more experienced, this guide will probably not be suitable for you. I will make analogies to Python, but you can ignore them if you don’t know that language.

First Program

For your first program, go to the site https://replit.com/@mattox/CPP-First#main.cpp .

You will see three panes. The left-most pane labeled Files, with a few icons on the side. There will be one file listed there, main.cpp. You should mouse-over the icons to see what they do. Click on the paper icon at the top to get back to the file list.

The second pane (in the middle) is the actual source code. It will have a small program in it, which we will talk about in a minute.

The last pane has two tabs, Console and Shell. They are almost identical in functionality.

If you click Run at the top, you will see this in the shell window: 

> make -s
> ./main
Hello World!
>

The > character is called the prompt, and allows you to input commands if you want. The command make builds the program for you. The command ./main runs the program.

Now let’s look at the program. 

1: #include <iostream>
2: 
3: int main() {
4:   std::cout << "Hello World!\n";
5: }

Unlike languages like Python, C++ does not include very many builtin functions. If you want to use something, you need to use a #include command, which is similar to the import command in Python. Line 1 includes the definitions for the iostream package. This allows us to do input and output operations.

Line 3 declares the main function. This is the function that will run when we start the program. The int means that this function will return an integer value. The convention is if main return the number 0 it means the program executed correctly. If it returns a non-zero, then some kind of error happened. We are not going to make much use of that.

The () part means that this function has no parameters. Finally, you see a { and a corresponding } two lines later. These are called “curly braces” and indicate where the function’s code begins and ends. (The code that belongs to a function is often called the body of the function.)

Finally, there is one line that tells the program to print out "Hello World!", the traditional thing to write to demonstrate a program in a new language. The std::cout says to use the cout (short for character output) stream An input or output channel is called a stream in the C++ world. , and the std:: part says that it is found in the standard namespace. A namespace is a way of collecting and separating variable and function names from each other. Two namespaces can each have a variable with the same name but still be able to tell them apart. It’s like two families with different family names, but each with a child named Ziggy. If two people are in the same family, they can say “Ziggy” without ambiguity. But if you mean the Ziggy from a different family you would use the family name and say “Ziggy Patel” or “Ziggy Choi” or something like that.

We don’t want to have to type std:: in front of things all the time, so try modifying the code to look like this: 

1: #include <iostream>
2: 
3: using namespace std;
4: 
5: int main() {
6:   cout << "Hello World!\n";
7: }

The using namespace std line means we can refer to anything in the standard namespace without needing to specify it. This would cause problems if you happened to have a variable named cout, but most people don’t do things like that.

Variables and Functions

There are three kinds of variables in C++.

Global variables
These are accessible throughout the program or module.
Local variables
These are defined inside of a function and exist only when a function is called.
Parameters
These variables are also part of a function and get their values from whoever calls the function.

Before a variable can be used it must be declared. A declaration has three parts: a type, a name, and (optionally) an initial value.

Global Variables

Consider this program. 

 1: #include <iostream>
 2: 
 3: using namespace std;
 4: 
 5: int answer = 42;
 6: double pi = 3.14159;
 7: 
 8: int main() {
 9:   cout << "The answer to the ultimate question is " << answer << endl;
10:   cout << "The number pi is approximately " << pi << endl;
11: }

There are two global variables, answer, which is an integer, and pi, which is a double-precision floating point number. I initialized them both at declaration time, but I could have done it in the body of main like this: 

 1: #include <iostream>
 2: 
 3: using namespace std;
 4: 
 5: int answer;
 6: double pi;
 7: 
 8: int main() {
 9:   answer = 42;
10:   pi = 3.14159;
11:   cout << "The answer to the ultimate question is " << answer << endl;
12:   cout << "The number pi is approximately " << pi << endl;
13: }

Special Warning: in C++, if you do not initialize a variable, then the contents will be “undefined”. The compiler will allocate memory for that variable, but it won’t clear that memory out first. If you try to use it, the contents might be 0, or it might be something completely random.

Functions

A function declaration has four parts: a return type, a name, a list of parameters, and a body.

Here is a simple function that adds two numbers together. 

 1: #include <iostream>
 2: 
 3: using namespace std;
 4: 
 5: double sum(double a, double b) {
 6:   double result;
 7:   result = a + b;
 8:   return result;
 9: }
10: 
11: int main() {
12:   double x,y;
13: 
14:   cout << "Give me two numbers: ";
15:   cin >> x >> y;
16:   cout << "The sum of " << x << " and " << y << " is " << sum(x,y) << endl;
17: }

In this program, a and b are called parameters. They serve as inputs to the function. When we call the function on line 16 the values x and y are called arguments. These two terms sometimes get used interchangeably.

The variable result is a local variable. In this code, we assign it the value a+b.

The return keyword says that the function is done and that the value provided should be the result of the function. Try running this function to see how it works.

This code could have been a lot simpler. First, we could have initialized result immediately, like this: 

1: double sum(double a, double b) {
2:   double result = a + b;
3:   return result;
4: }

And even then, since we only use that value one time, we could have returned a+b directly like this: 

1: double sum(double a, double b) {
2:   return a+b;
3: }

It’s not wrong to declare a variable and only use it once if it makes your program easier to read or understand, but avoid it if it’s not necessary.

Exercises:

  • Modify the program to take the sum of three numbers instead of only two. There are several ways you could go about this.
  • Add another function to do subtractions.
  • If you add the line #include <bits/stdc++.h> to the top in place of the #include <iostream> you will gain access to the sqrt function, that (you guessed it!) takes the square root of a number. Write a function that gives you the length of the hypotenuse of a triangle.

Conditions

Programs need to make decisions in order to do anything interesting. The most common way to do this is with the if statement. An if statement has three parts: a condition, a then-branch, and (optionally) an else-branch.

Here’s an example: 

 1: #include <bits/stdc++.h>
 2: 
 3: using namespace std;
 4: 
 5: double max(double a, double b) {
 6:   if (a>b) {
 7:     return a;
 8:   } else {
 9:     return b;
10:   }
11: }
12: 
13: int main() {
14:   double x,y;
15: 
16:   cout << "Give me two numbers: ";
17:   cin >> x >> y;
18:   cout << "The max of " << x << " and " << y << " is " << max(x,y) << endl;
19: }

Sometimes you don’t need the else branch. Here’s an example: 

 1: #include <bits/stdc++.h>
 2: 
 3: using namespace std;
 4: 
 5: int main() {
 6:   double x;
 7: 
 8:   cout << "Give me a number: ";
 9:   cin >> x;
10: 
11:   if (x<0)
12:     x=-x;
13: 
14:   cout << "The absolute value is " << x << endl;
15: }

A few syntax notes:

  • The condition part is always in parenthesis.
  • There is no then keyword, but there is an else keyword.
  • You use curly braces to contain the then-branch and the else-branch code.
  • The curly braces are optional if the code has only one statement it in.

Loops

Sometimes you want to do something more than once. Loops are one way to accomplish this. There are two kinds of loops you should know.

While loops

A while loop has a condition and a body. When it runs, it checks if the condition is true. If not, it will exit. If it is, then it will run the body and then try again.

Here is the famous factorial function using while loops. 

 1: #include <bits/stdc++.h>
 2: 
 3: using namespace std;
 4: 
 5: int fact(int n) {
 6:   int result = 1;
 7: 
 8:   while (n>1) {
 9:     result *= n;
10:     --n;
11:   }
12: 
13:   return result;
14: }
15: 
16: int main() {
17:   int x;
18: 
19:   cout << "Give me a number: ";
20:   cin >> x;
21: 
22:   cout << x << "! = " << fact(x) << endl;
23: }

Here is a recursive version; we will talk about recursion later. 

1: int fact(int n) {
2:   if (n==0)
3:     return 1;
4: 
5:   return n * fact(n-1);
6: }
